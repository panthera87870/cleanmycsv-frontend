Comprendre le nettoyage de vos CSV

Pourquoi nettoyer un CSV ?

Un fichier CSV peut sembler simple : des lignes, des colonnes, des valeurs.
Mais en réalité, chaque export a sa propre manière d’écrire les dates, les nombres ou les séparateurs.

Ces variations créent des incohérences invisibles qui :

  • faussent les calculs ou l'analyse de données,
  • cassent les imports dans Excel ou un logiciel comptable,
  • ralentissent le travail manuel de correction,
  • ou bloquent une automatisation critique.

CleanMyCSV organise et standardise ces données pour que vous puissiez travailler sans friction.

Comment fonctionne CleanMyCSV ?

Étape 1 — Analyse du fichier

Lorsque vous déposez votre CSV, CleanMyCSV commence par comprendre sa structure.

Détection automatique de l'encodage

Tous les fichiers ne sont pas encodés de la même manière (UTF-8, ISO-8859-1, Windows-1252…).
CleanMyCSV utilise chardet pour repérer l’encodage puis icon-lite pour convertir proprement le fichier.

Cela évite :

  • les caractères “�”,
  • les accents illisibles,
  • les colonnes décalées.

Détection du séparateur réel

Les CSV peuvent utiliser : virgule, point-virgule, tabulation, pipe…

CleanMyCSV teste les structures possibles pour choisir le bon séparateur, même si :

  • le fichier n’a pas d’extension,
  • il a été exporté d’un logiciel exotique,
  • il mélange plusieurs types.

Résultat : le fichier est prêt à être analysé sans erreur.

Avant 
date, montant; description , categorie
04/12/23; 1.200,50; "Achat test" ; frais
Après 
date;montant;description;categorie
04/12/2023;1200.50;Achat test;frais

Étape 2 — Lecture et validation des lignes

Le fichier est ensuite décodé et analysé ligne par ligne via PapaParse, un parseur très robuste.

CleanMyCSV vérifie :

  • le nombre de colonnes attendu,
  • les lignes vides,
  • les champs décalés,
  • les valeurs non conformes.

Toutes les irrégularités sont consignées dans un rapport.

Étape 3 — Normalisation des données

C’est ici que CleanMyCSV agit vraiment comme un “nettoyeur”.

Normalisation des dates (normalizeDate)

Les dates sont souvent le problème n°1 des CSV.

CleanMyCSV détecte le format, puis :

  • convertit la date dans un format unifié,
  • enlève l’heure inutile,
  • corrige les inversions jour/mois quand c’est détectable,
  • repère les dates impossibles (ex. “32/20/2040”).
Avant 
02/01/23
2023-02-01 00:00:00
02.01.2023
1 Feb 2023
Après 
2023-02-01
2023-02-01
2023-02-01
2023-02-01

Normalisation des montants (normalizeAmount)

Les montants peuvent venir sous des formes multiples.

CleanMyCSV :

  • supprime les espaces parasites,
  • détecte automatiquement la convention utilisée,
  • unifie les décimales,
  • detecte et ajoute une colonne devise,
  • convertit tout en un format numérique cohérent.

Résultat : tous les montants sont stables et exploitables.

Avant 
1.200,50
€1 200.50
- 1200,5
1200
Après 
1200.50
1200.50;€;
-1200.50
1200.00

Nettoyage structurel

CleanMyCSV reconstruit chaque ligne de façon cohérente :

  • colonnes alignées
  • guillemets retirés lorsqu’ils ne sont pas nécessaires
  • lignes vides supprimées
  • champs incomplets signalés
  • corrections regroupées dans le rapport
Exemple de correction : champ manquant
Avant 
date;montant;catégorie
04/12/23;1200.50
Après 
date;montant;catégorie
04/12/2023;1200.50;""
Exemple de correction : doublon silencieux
Avant 
04/12/2023;1200.50;frais
04/12/2023; 1200.50 ; frais
Après 
04/12/2023;1200.50;frais

Étape 4 — Rapport détaillé

Une fois le fichier nettoyé, CleanMyCSV génère un rapport clair contenant :

  • nombre de lignes analysées
  • nombre d’erreurs trouvées
  • types d’erreurs détectées
  • valeurs corrigées
  • format final choisi

Le rapport ne cherche pas à être “technique pour les experts”, mais compréhensible pour tous, y compris si vous n’avez jamais touché un CSV auparavant.

Étape 5 — Export propre et cohérent

Le fichier final est généré avec :

  • un encodage propre (UTF-8)
  • un séparateur stable
  • des valeurs harmonisées
  • des colonnes alignées
  • des montants normalisés
  • des dates cohérentes

Vous obtenez un CSV immédiatement exploitable dans Excel, Google Sheets, un logiciel comptable, ou dans un script automatisé.

Sécurité des données et confidentialité

Tout fichier déposé est supprimé après traitement.
Aucune donnée n’est conservée.
Aucun fichier n’est analysé à des fins externes.

Pour en savoir plus sur la protection de vos données, veuillez consulter notre Politique de Confidentialité.
